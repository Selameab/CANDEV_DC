# CANDEV Data Challenge

Install requirements
-----
```sh
$ pip install -r requirements.txt
```

Getting started on Linux/Mac
-----
Start Scrapers
```sh
$ bash ./start_scrapers.sh
```
Start Server
```sh
$ bash ./start_server.sh
```

Getting started on Windows
-----
Start Scrapers
```sh
$ start_scrapers.bat
```
Start Server
```sh
$ start_server.bat
```
